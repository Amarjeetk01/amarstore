# E-commerce Application (AmarStore)

This is an e-commerce application with both a frontend and backend. The project is divided into two parts: the frontend, built with React, and the backend, built with Node.js and Express.

## Backend

The backend of this project is built with Node.js and Express. It provides API endpoints for handling user authentication, managing products, brands, categories, user information, and shopping carts. It also integrates with MongoDB as a database and Stripe for payment processing.

### Prerequisites

Before running the backend, make sure you have the following dependencies installed:

- Node.js
- MongoDB
- Stripe API Key (You need to set it up in a .env file)

## Frontend

The frontend is built with React and is designed to interact with the backend API. It provides a user-friendly interface for browsing products, adding them to the cart, and completing the checkout process.

## Usage

- Visit http://localhost:3000 in your web browser to access the frontend.
- You can use the API endpoints provided by the backend for various operations.

- `backend/`: Backend code.
- `frontend/`: Frontend code.
